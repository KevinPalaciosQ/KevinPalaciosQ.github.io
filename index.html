Container Manager

A comprehensive container management system that provides deep insights into Linux container operations, utilizing kernel modules, bash scripting, and Rust for advanced system monitoring and container lifecycle management.

Built using a multi-technology approach, the project combines C kernel modules, Rust services, and Docker to create a robust system for generating, tracking, and analyzing container resources in real-time.

Key features include:

  • Automatic container generation with resource-based categorization Dynamically creates 10 containers with high and low resource consumption profiles using a cronjob
  • Real-time monitoring of container memory and CPU usage Captures detailed metrics including total RAM, free memory, and process-level resource utilization
  • Kernel-level metrics collection and logging Uses Linux kernel modules to extract comprehensive system and container information stored in /proc
  • Dynamic container management and elimination Intelligent container lifecycle management with sorting and elimination based on resource usage
  • JSON-based system information reporting Generates structured logs with detailed process information, including PID, container ID, and resource consumption

The system leverages Rust's memory safety, C kernel modules' low-level access, and Docker's containerization to create a powerful, flexible container management solution that provides unprecedented visibility into system resources and container operations.

USAC Linux

This project enhances the Linux kernel by implementing custom system calls and kernel modules, focusing on security functionalities and system performance monitoring. It integrates custom syscalls to capture memory snapshots, track system call usage, and retrieve I/O statistics.

The custom syscalls are designed to provide real-time insights into memory usage, system call activity, and I/O throttling, helping developers and system administrators monitor and optimize system performance efficiently.
